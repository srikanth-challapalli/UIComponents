Using multiple fonts like (Courier New', Courier, monospace) in a single font-family attribute says by default apply Courier New if not available apply Courier font .If both are not available then apply monospace.
Another important thing to note, if we apply styling to entire body and another styling is applied to certain area of the html page, then styling for entire body will be ignored and specific styling will only be applied
Another important thing is we can import fonts from external entities like google or any other styling websites. Search for google fonts in google and then copy the specific link for that specific font and include it in html page .Then configure the font in css file to reflect the font changes in the page as shown below for heading "Sample Heading" the font is taken from google and added the link in html and configured corresponding font family in respective css file. Another way is to copy the import link provided in google fonts "@import url('https://fonts.googleapis.com/css2?family=Akaya+Telivigala&display=swap');" directly in css then we dont have to provide the link tag in html.
