If you would like to apply any styling for entire page then we use universal selection . The systax is pretty simple "*{}" inside the folwer brackets anything you put that will be applicable for entire page in which the respective stylesheet file is included. Normally this tag will be used for fonts so that the page will be looking symmetric. In our example I applied style to change the entire page text color to blue. However, if someone wish to override this they would be able to do it by specifying the selector and provide the styling. In this example even we applied the universal selector as to change the text color to blue.
