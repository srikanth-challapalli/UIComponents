Inline Element

Inline elements will always starts with new line and it occupies only certain width of the webpage.Example for Inline elements are
Inline Elements
We cannot change the width and height of the inline elements
Inline elements will not have margin-top and margin-bottom properties like block level element. They will only have marging-left and margin-right
We can convert inline elements to block level by changing the style property display :block in css file to the inline html field.
